# MelodyHub – Music Search Web Application

**Overview:**

MelodyHub is a responsive music search web application built using **HTML, CSS, and JavaScript** that integrates the **iTunes Music Search API** to provide a seamless music discovery experience. The application allows users to search for songs or artists, view album artwork, artist information, album details, genre, and release year, and listen to 30-second song previews directly within the browser.

The application features a modern and responsive user interface with smooth animations, dynamic search functionality, and real-time data retrieval using the Fetch API. It also includes a single-audio playback feature, ensuring that only one song preview plays at a time for an improved user experience.

**Key Features:**

* Search songs and artists using the iTunes Music Search API.
* Display song title, artist, album, genre, release year, and album artwork.
* Play 30-second audio previews directly in the application.
* Open songs in Apple Music through a direct link.
* Responsive design compatible with desktop and mobile devices.
* Loading animation while fetching search results.
* "No Songs Found" message for invalid searches.
* Single audio playback to prevent multiple songs from playing simultaneously.
* Search using both the Search button and the Enter key.

**Technologies Used:**

* HTML5
* CSS3
* JavaScript (ES6)
* Fetch API
* iTunes Music Search API
